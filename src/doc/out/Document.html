Class: Document

Document(doc, data, article, topic, ind, number)

The document class. Upon it's instatiation it creates an object for an article of the representive topic. It is created by calling the constructor as: >>> var article = new Document(articel-reference, data, article-id, topic-id, topic-index, article-index) The article class saves all of it#s mentions and marks it's text.
