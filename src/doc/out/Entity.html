Class: Entity

Entity(entity, topic, entity_name, identifier, number)

The entity class. A new object of the entity class can be created by calling the cosntructor: >>> var entity = new Entity(entity, topic, entity_name, identifier, number) The entity class also instantiates each of its' mentions.
